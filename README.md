# Sistema de Asistencia con QR

Sistema desarrollado por Joswii en Python con Streamlit para registrar asistencia de estudiantes mediante códigos QR.

## Funciones

- Registro de estudiantes
- Generación de QR
- Escaneo de QR
- Registro manual de asistencia
- Visualización de registros

## Ejecutar el sistema

Instalar librerías:

pip install -r requirements.txt

Ejecutar aplicación:

streamlit run asist.py
